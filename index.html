DWA: Distributed Web Assets

Our goal is to create the new kind of web-development platform.

How it works in a few words:
  • Every DWA entity - is a JavaScript module (ESM) with a single default export.
  • Each module has it's own URI and physically placed in IPFS.
  • Each module is accessible via serverless endpoint (cloud function) with a parameters support.
  • Each endpoint caches the request result as the static file in CDN.

So, to build your page or application you need to chain some DWAs in a row. That's it.
